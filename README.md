# Crime data visualization

A short Python project for visualising crime data collected by the Metropolitan Police, UK.
Data was downloaded from https://data.police.uk/data/


## Installation

Written in Python 2.7.

Modules and Packages:

* Pandas
* Numpy
* Matplotlib
* re
* Bokeh


## Usage

Run the script from terminal, e.g.

```bash
python DataVisualization.py
```
